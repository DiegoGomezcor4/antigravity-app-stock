# Guía de Configuración: Base de Datos (Supabase)

Para persistir tus datos en la nube y acceder desde múltiples dispositivos (celular, tablet, PC), usaremos **Supabase**.

## Pasos para Crear tu Proyecto

1. **Crear Cuenta**: Ve a [supabase.com](https://supabase.com) y regístrate (es gratis).
2. **Nuevo Proyecto**:
   - Haz clic en **"New Project"**.
   - Elige tu organización.
   - Nombre: `StockApp` (o el que quieras).
   - Base de datos: Genera una contraseña segura.
   - Región: Elige la más cercana (ej: Sao Paulo o US East).
   - Haz clic en **"Create new project"**.
3. **Obtener Credenciales**:
   - Una vez creado, ve a **Settings** (ícono de engranaje) -> **API**.
   - Necesitarás dos valores:
     - `Project URL`
     - `anon public` Key

## Cuando tengas estos datos
Avísame o copialos en el chat. Yo configuraré la aplicación para que deje de usar el navegador y empiece a guardar todo en tu nueva base de datos.
